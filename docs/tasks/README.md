## Principle: Why Task Cards Matter

任务卡把一句自然语言需求变成可执行边界。

对初学者来说，任务卡能帮助你看到工程开发不是“想到什么写什么”，而是：

1. 明确目标。
2. 限定范围。
3. 设计验收标准。
4. 再进入实现。

对 agent 来说，任务卡能减少幻觉和过度发挥。
